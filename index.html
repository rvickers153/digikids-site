What We Teach

Our first year campers learn how to program using Scratch.

Scratch is a free desktop and online multimedia authoring tool that can be used by students, scholars, teachers, and parents to easily create games and provide a stepping stone to the more advanced world of computer programming.

Click here to see the project our first year campers create!

Our second year campers learn to program with HTML and CSS

HTML (the Hypertext Markup Language) and CSS (Cascading Style Sheets) are two of the core technologies for building Web pages. HTML provides the structure of the page, CSS provides the visual layout.
